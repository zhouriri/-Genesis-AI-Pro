# SOP

## 步骤 1：获取 FAQ 内容

使用以下命令获取 FAQ 文档内容：

```bash
curl -s "https://docs.coze.cn/api/open/docs/tutorial/openclaw_faq"
```

## 步骤 2：分析用户问题

根据用户的问题类型，在 FAQ 文档中查找对应的解决方案：

| 问题类型 | FAQ 章节 |
|---------|---------|
| 什么是 OpenClaw | "了解概念" -> "什么是 OpenClaw" |
| 模型配置问题 | "使用方法" -> "如何修改模型" |
| 积分消耗过快 | "使用方法" -> "积分消耗太快了怎么办" |
| 安装技能 | "使用方法" -> "如何为 OpenClaw 安装技能" |
| 飞书集成问题 | "飞书相关" |
| 服务启动失败 | "故障修复" -> "界面一直显示预览中" |
| 消息无响应 | "故障修复" -> "OpenClaw 不回应飞书消息" |
| 401 错误 | "故障修复" -> "对话提示 401 Unauthorized" |

### 步骤 3：提供解决方案

根据 FAQ 内容，为用户提供：

1. **问题描述**：简要说明可能的原因
2. **解决步骤**：具体的操作指南
3. **相关命令**：可执行的命令示例
4. **参考链接**：相关文档或资源

如果用户确认帮助排查定位，则按照操作步骤进行执行
