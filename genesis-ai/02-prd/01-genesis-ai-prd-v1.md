## 一、产品概述 (Product Overview)

### 1.1 产品定位

**一句话描述：**
> Genesis AI 是一个基于 Web3 的 AI 驱动合成资产量化平台，用户可通过 USDC 一键投资加密货币、美股、黄金及大宗商品，无需 KYC，且由 AI Agent 自主执行交易策略。

**核心价值主张：**

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│   任何人在任何地方，用 USDC，就能投资全球资产         │
│                                                     │
│   🚫 无需 KYC    🚫 无需开户    🚫 无需托管         │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 1.2 解决的问题

| 用户痛点 | Genesis AI 解决方案 |
|:--|:--|
| 17亿人无法开设美股账户 | 钱包地址即可投资美股 |
| DeFi 门槛高，小白不会操作 | AI Agent 全自动执行策略 |
| 稳定币持有者资产闲置 | USDC 直接生息，投资全球资产 |
| 中心化 AI 平台不透明 | 去中心化架构，用户自持私钥 |
| gTrade 等界面太复杂 | RockFlow 式极简 UI |

### 1.3 已确认的商业决策

| 决策项 | 选择 | 说明 |
|:--|:--|:--|
| **注册主体** | 🇸🇬 新加坡 | 亚洲金融中心，合规友好，亚洲用户信任度高 |
| **目标市场** | 全球（除美、英、中） | 规避强监管市场，专注新兴市场用户 |
| **Token 规划** | 待定 | MVP 阶段专注产品，Token 规划后续讨论 |

## 五、用户旅程 (User Journey)

### 5.1 主流程：5 分钟从 0 到投资

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│          │    │          │    │          │    │          │    │          │
│  连接    │ -> │  充值    │ -> │  选择    │ -> │  AI      │ -> │  查看    │
│  钱包    │    │  USDC   │    │  策略    │    │  自动交易 │    │  收益    │
│          │    │          │    │          │    │          │    │          │
└──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘
    10秒          30秒          30秒          1次授权          实时
```

### 5.2 详细步骤

**Step 1：连接钱包**
```
- 用户点击 "Connect Wallet"
- 支持：MetaMask / WalletConnect / Coinbase Wallet / Safe{Wallet}
- 展示钱包地址，链自动切换到 Polygon zkEVM
```

**Step 2：充值 USDC**
```
- 展示 Polygon zkEVM 上的 USDC 充值地址
- 或直接从 CEX 链上转账
- 最低充值：$10 USDC
```

**Step 3：选择 AI 策略**
```
- 三种预设策略：
  🌊 保守型（收益 3-8%/月，波动小）
  ⚖️ 平衡型（收益 8-15%/月，波动中）
  🚀 激进型（收益 15-30%/月，波动大）
  🔧 自定义（用户配置参数）

- 点击查看策略详情（历史回测、风险评级）
- 选择后点击 "Activate AI"
```

**Step 4：授权 Session Key**
```
- 弹窗展示授权内容：
  "授权 Genesis AI Agent 最多操作 $X,XXX USDC"
  "仅限现货交易，禁止合约杠杆"
  "可随时撤销授权"

- 用户确认并签名
- 签名即完成授权
```

**Step 5：AI 开始交易**
```
- 用户无需任何操作
- AI Agent 7/24 自动执行策略
- 实时推送交易通知（Telegram/Discord）
- 随时在 Dashboard 查看收益
```

### 5.3 Telegram Bot 用户流程

**Phase 1 基础功能：**
```
/start           - 开始，连接钱包
/strategies      - 查看可用策略
/activate [type] - 激活策略（保守/平衡/激进）
/portfolio       - 查看当前组合
/withdraw        - 提取 USDC
/help            - 帮助
```

**Phase 2 高级功能：**
```
自然语言对话：
"激活保守型策略"      → 激活策略
"我的收益怎么样？"      → 查询收益
"换成激进型策略"      → 切换策略
"帮我止损"            → 执行风控
```
